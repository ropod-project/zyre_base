# Zyre Base classes

## zyre_base
Abstract Base Class to enable comfortable zyre communication through inheritance.

## Install
```
git clone git@git.ropod.org:ropod/ropod_common.git
sudo mv ropod_common /opt/ropod/
```

## pyre_base
For the pyre base class, you can run:

```
sudo pip3 install -e .
```
